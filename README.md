# urovo-barcode-sdk

Urovo Barcode Reader Wrapper

## Installation

```sh
npm install urovo-barcode-sdk
```

## Usage

```js
import { multiply } from 'urovo-barcode-sdk';

// ...

const result = await multiply(3, 7);
```
